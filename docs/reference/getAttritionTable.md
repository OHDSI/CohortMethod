# Get the attrition table for a population

Get the attrition table for a population

## Usage

``` r
getAttritionTable(object)
```

## Arguments

- object:

  Either an object of type
  [CohortMethodData](https://ohdsi.github.io/CohortMethod/reference/CohortMethodData-class.md),
  a population object generated by functions like
  [`createStudyPopulation()`](https://ohdsi.github.io/CohortMethod/reference/createStudyPopulation.md),
  or an object of type `outcomeModel`.

## Value

A `tibble` specifying the number of people and exposures in the
population after specific steps of filtering.
