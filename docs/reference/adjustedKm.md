# Compute a weight-adjusted Kaplan-Meier curve

Compute a weight-adjusted Kaplan-Meier curve

## Usage

``` r
adjustedKm(weight, time, y)
```

## Arguments

- weight:

  Vector of observation weights

- time:

  Vector of event times

- y:

  Vector outcomes (0 indicates censoring, 1 indicates event-of-interest)
