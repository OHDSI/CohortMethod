# Get the default table 1 specifications

Loads the default specifications for a table 1, to be used with the
[`createCmTable1`](https://ohdsi.github.io/CohortMethod/reference/createCmTable1.md)
function.

Important: currently only works for binary covariates.

## Usage

``` r
getDefaultCmTable1Specifications()
```

## Value

A specifications objects.
