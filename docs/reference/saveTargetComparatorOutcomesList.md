# Save a list of `TargetComparatorOutcomes` to file

Write a list of objects of type `TargetComparatorOutcomes` to file. The
file is in JSON format.

## Usage

``` r
saveTargetComparatorOutcomesList(targetComparatorOutcomesList, file)
```

## Arguments

- targetComparatorOutcomesList:

  A list of objects of type `TargetComparatorOutcomes` as created using
  the
  [`createTargetComparatorOutcomes()`](https://ohdsi.github.io/CohortMethod/reference/createTargetComparatorOutcomes.md)
  function.

- file:

  The name of the file where the results will be written
